# ``AnalyticsLibrary``

Firebase Analytics integration with type-safe event tracking.

## Overview

AnalyticsLibrary provides a protocol-oriented analytics solution with built-in Firebase Analytics support.
It offers type-safe event definitions, automatic session tracking, and SwiftUI integration for seamless analytics implementation.
